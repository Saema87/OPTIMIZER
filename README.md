## 🚀 Deploy (Free, 5 Minutes)

### 1. Get Free Groq API Key
→ [console.groq.com](https://console.groq.com) · No credit card needed

### 2. Push to GitHub
```bash
git init
git add .
git commit -m "Prompt Optimizer - Glassmorphism"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/prompt-optimizer.git
git push -u origin main
```

### 3. Deploy on Streamlit Cloud
1. Go to [share.streamlit.io](https://share.streamlit.io) → Sign in with GitHub
2. Click **"New app"** → Select your repo → File: `app.py`
3. **Advanced settings → Secrets** → Add:
   ```toml
   GROQ_API_KEY = "gsk_your_key_here"
   ```
4. Click **Deploy** → Live at `your-app.streamlit.app`
